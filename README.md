# Anonim Oylama Sistemi (AOS)

Kullanıcıların özel odalarda anonim oylama yapabildiği web uygulaması.

## Özellikler

- **Anonim Oylama:** Oyların kimden geldiği tespit edilemez
- **Oda Sistemi:** Link veya kod ile özel oylama odaları
- **Adaylık:** Katılımcılar kendilerini aday gösterebilir
- **Canlı Sonuçlar:** Oylama bitiminde grafiklerle görüntüleme
- **Tek Oy Güvenliği:** "Bir kişi, bir oy" kuralı

## Teknoloji Stack

- **Backend:** Go
- **Frontend:** Next.js (React + TypeScript)
